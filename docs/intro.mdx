# Welcome!

Welcome to Orchid!

Orchid is a ever-changing NFT collection that is hosted on Polygon. Each NFT is connected to a geolocation and the image is generated based on 3 factors within this location:
- Air Quality
- Water Quality
- Fauna Diversity

If nature becomes better in the area, you're NFT will be come more vibrant and more value, while if the natures dies, the image becomes more sober.

To learn more about this collection, continue reading!
